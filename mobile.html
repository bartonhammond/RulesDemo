Mobile information

I've built a simple mobile app using Application Craft which utilizes JQuery Mobile and PhoneGap to build a mobile deployed application.

It makes JSONP requests to RulesDemo web application's webservice and displays the results.

Install on Android

Android QR code After scanning the QR Code, it will be downloaded by your selected browser. Then using the Android application "My Files", go to the download direction and install the app.

Download I use Dropbox to install on my Android - just install Drop box on both your PC and Android and then just copy the "apk" to your Dropbox on the PC and then on the Android, use Dropbox and click the "apk" to install.
